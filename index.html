Things I've Designed

I've worked on Game Design, Graphic Design, and Web Design projects for a range of different interests and clients. Here's a small selection of some of my recent projects. Reach out if you have any questions about these projects.

A Community of Friends

A collaborative board game centered around creating a community around a character struggling through a number of events. Each player has a choice of types of support to give and must discuss together the best course of action for the problem at hand.

This game was created as a group assignment in my Master's Degree Program

Roles: Co-lead Designer, Graphic Designer

Instructional Design: The Game

An easy to play board game based on the instructional design process. Similar in play-style to many of th games of childhood, players find themselves trying to work their way through a full course development - but not without some struggles along the way!

This game was created as an update on a previous draft from work. I designed the elements and had it professionally printed.

Roles: Graphic Designer, Developer

Story Cache

An augmented reality experience designed to answer bring communities together. It uses the camera on your phone to recognize specific photos strategically placed around a defined area. Scanning that image will bring forth a short audio story of an important memory related to that photo.

This experience was created as a group assignment in my Master's Degree Program.

Roles: Designer, Narrative Lead, Developer

HCA 705 Emergency Management Scenario

A 7-week long scenario where students must make choices as they take on the role of an emergency manager. At some point, they know they'll run into some sort of an incident, so the choices in the build-up are just as important as those during an incident. This experience is taken in tandem with coursework to allow students to practice their decision-making skills.

This was created for a course at the University of New England.

Roles: Designer, Narrative Lead, Developer, Asset Creator

In Progress: Assessment Crafting Aid

A digital, interactive version of a previouly designed infographic. The aim of this tool is to help subject matter experts and instructors to develop well-aligned assessments. It guides the user step-by-step through the backwards design process and provides a full synopsis of the well-crafeted assignment instructions.

This project is still in active development. It is being designed with the University of New England

Roles: Designer, Developer

In Progress: Media Tool Menu

As instructional designers, we use many different tools to create the interactive media used in courses. Sometimes the choices of tools can be overwhelming, so the aim of this tool is to alleviate that feeling. The user simply has to select the type of project and the complexity level they would like, and the tool will provide a number of options available for them to use.

This project is still in active development. It is being designed for the University of New England.

Roles: Designer, Developer, Contributor
